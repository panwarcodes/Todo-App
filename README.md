# Todo App

My first ever JavaScript project built completely from scratch — no tutorials used.

It took around 4 days with 3–4 hours of work daily, along with multiple brain explosions xD.

Since this is my first real JavaScript project, I felt it deserved its own separate repository.

This is Version 1 of the project.

Features:
- Add tasks
- Edit tasks
- Delete tasks
- LocalStorage support

Things I learned while building this:
- DOM manipulation
- Event listeners
- Forms and inputs
- Dynamic element creation
- LocalStorage
- Handling app state
